# Changelog

## Unreleased

- 补充 Marketplace 搜索关键词，强化 `摸鱼 / 上班摸鱼 / 贴吧阅读器 / 老板键` 等检索命中。
- 调整首页和首次引导，优先支持先试读，再按需补登录态、Python 和 `aiotieba`。
- 左侧新增 `收藏` 视图，并把原来的 `最新` 明确改名为 `最近`。

- 网页回退链路下，楼中楼改为明确的“仅预览”提示，不再误导成可完整展开。
- `紧凑模式 / 低存在感模式` 现在会真实影响吧页和帖子页的阅读样式。

## 0.0.5

- 新增更紧凑的首页入口，统一开始使用路径。
- 新增更人话的状态提示，首页、状态栏和环境诊断口径保持一致。
- 统一吧页、帖子页、只看楼主和楼中楼的加载反馈。
- 调整命令面板分层，按 `开始使用 / 阅读 / 维护` 组织入口。
- 强化 `关注吧 / 最新 / 历史` 的空态引导。
- 新增帖子页快捷键帮助。
- 新增状态栏主题预设切换，支持 `默认 / 极简 / 文档风`。
- 更新 README，加入新的主界面截图，并改成面向使用者的产品说明。

## 0.0.4

## 0.0.1

- 初版发布。
- 支持 `关注吧 / 最新 / 历史` 三个视图。
- 支持帖子阅读器、楼中楼分页、图片缩略图和大图预览。
- 支持 `继续阅读`、`浏览指定链接`、`同步我关注的贴吧`。
- 支持首次引导、环境诊断、`aiotieba` 一键安装。
- 支持 `Ctrl+Alt+X` 老板模式。
