# Changelog

## 0.0.5

- 新增更紧凑的首页入口，统一开始使用路径。
- 新增更人话的状态提示，首页、状态栏和环境诊断口径保持一致。
- 统一吧页、帖子页、只看楼主和楼中楼的加载反馈。
- 调整命令面板分层，按 `开始使用 / 阅读 / 维护` 组织入口。
- 强化 `关注吧 / 最新 / 历史` 的空态引导。
- 新增帖子页快捷键帮助。
- 新增状态栏主题预设切换，支持 `默认 / 极简 / 文档风`。
- 更新 README，加入新的主界面截图，并改成面向使用者的产品说明。

## 0.0.4

- 修复 Marketplace README 截图无法显示的问题，改为使用 GitHub 绝对图片地址。

## 0.0.3

- 修复老板模式错误地依赖用户工作区路径的问题，伪装文件现在来自扩展自带内容。
- 修复 `只看楼主` 页码显示不真实的问题，切换后改为请求真实的楼主分页数据。
- 更新 README，改成面向使用者的说明，并补充功能截图与 `aiotieba` 鸣谢。

## 0.0.2

- 修复正式安装包误带 `aiotieba-master` 开发目录的问题。
- 修复 `aiotieba` 导入失败时误导用户走本地源码编译的问题。
- 调整插件内一键安装逻辑，优先走 PyPI 预编译 wheel。
- 补充安装失败场景的说明文档和排障提示。

## 0.0.1

- 初版发布。
- 支持 `关注吧 / 最新 / 历史` 三个视图。
- 支持帖子阅读器、楼中楼分页、图片缩略图和大图预览。
- 支持 `继续阅读`、`浏览指定链接`、`同步我关注的贴吧`。
- 支持首次引导、环境诊断、`aiotieba` 一键安装。
- 支持 `Ctrl+Alt+X` 老板模式。
