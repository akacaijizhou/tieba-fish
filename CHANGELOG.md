# Changelog

## Unreleased

## 0.0.8

- 首页和首次使用流程改成面向普通用户的 `开始看帖`，优先支持输入吧名、粘贴帖子链接和继续阅读。
- 命令面板入口重新收敛到 `Tieba Fish / 阅读 / 维护`，隐藏 Cookie、Python 下载和 `aiotieba` 等低层调试入口。
- 新增 `维护: 完全重置并重载`，可一键清空本地贴吧登录、关注吧、收藏、历史、缓存和首页引导状态。
- 缺少 Python 时默认提示安装；本机已有 Python 但缺少阅读增强组件时，默认自动安装 `aiotieba`。
- 优化 Python 安装检测，优先识别真实安装目录和 `py`，避开 WindowsApps 的 `python` 占位符。
- 调整 Windows 上 Python 自动安装参数，不再默认安装 Python Launcher，避免被系统安装策略拦截后整体回滚。
- 帖子页和吧页失败态移除 `VS Code 浏览器 / 系统浏览器` 回退按钮，改为 `重试 / 环境诊断`。
- 楼中楼正文现在展示 `某某 回复 某某` 的被回复对象，并统一回复者、被回复者和“回复”文案的视觉层级。
- 网页回退链路下的楼中楼提示改为引导检查 Python 和阅读增强组件，不再推荐跳转浏览器。
- README 更新为下载后如何开始、阅读增强组件、检查问题和完全重置等面向用户的说明。

## 0.0.7

- 补充 Marketplace 搜索关键词，强化 `摸鱼 / 上班摸鱼 / 贴吧阅读器 / 老板键` 等检索命中。
- 调整首页和首次引导，优先支持先试读，再按需补登录态、Python 和 `aiotieba`。
- 左侧新增 `收藏` 视图，并把原来的 `最新` 明确改名为 `最近`。
- 帖子页新增收藏入口，并把失败态回退入口收拢到环境诊断。
- `继续阅读` 现在会恢复 `只看楼主` 等阅读上下文。
- 网页回退链路下，楼中楼改为明确的“仅预览”提示，不再误导成可完整展开。
- `紧凑模式 / 低存在感模式` 现在会真实影响吧页和帖子页的阅读样式。
- 主题系统升级为 `预设 / 密度 / 对比` 三轴，并统一首页、诊断页、快捷键页和阅读页的主题层。
- README 截图统一收拢到静态资源目录，并更新文档链接。
- 将帖子页“回到顶部”改成图标按钮，并避开底部翻页操作区。
